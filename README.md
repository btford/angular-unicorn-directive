# Angular Unicorn Directive

## Usage
1. Include `unicorn.js`.
2. Add `unicorn-directive` as a dependency to your app.
3. Make `<unicorn>`s.

## Example

```html
<div ng-app="unicorn-directive">
  <unicorn></unicorn>
  <p>^ This is a unicorn</p>
</div>
</pre>
```
